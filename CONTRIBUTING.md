# Contributing

Thanks for helping improve OptiShield. Contributions should preserve the project's privacy-first, ethical scope.

## Principles

- No telemetry, analytics, accounts, tracking, fingerprinting, or remote code.
- Request only permissions that are necessary for optical privacy features.
- Prefer subtle, accessible perturbations over aggressive distortion.
- Keep benchmark tooling local-only.

## Workflow

1. Install dependencies with `npm install`.
2. Run `npm run typecheck` and `npm run build` before opening a PR.
3. Document user-facing changes in `README.md` or `docs/`.
4. Include accessibility and performance considerations in PR descriptions.
