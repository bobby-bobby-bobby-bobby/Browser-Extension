# Security Policy

## Reporting vulnerabilities

Please report security issues privately to the maintainers. Do not open public issues for exploitable vulnerabilities.

## Scope

OptiShield is a browser extension that affects visual rendering only. It does not provide cryptographic protection and does not protect against digital capture, malware, spyware, browser compromise, or screenshots.

## Privacy boundaries

The extension must not collect telemetry, fingerprint users, transmit browsing data, execute remote code, require accounts, or include analytics.
